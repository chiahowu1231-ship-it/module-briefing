# MODULE INDUSTRY BRIEFING

全球模組產業日報 — 5G / RF / IoT / ODM 供應鏈每日情報

## 部署

替換 2 個檔案到 GitHub Repo：

| 檔案 | 路徑 |
|------|------|
| `module_briefing.py` | 根目錄 `/module_briefing.py` |
| `module_briefing.yml` | `.github/workflows/module_briefing.yml` |

Secrets (`Settings → Secrets → Actions`)：
`GEMINI_API_KEY` / `GMAIL_USER` / `GMAIL_APP_PASSWORD` / `GMAIL_BCC`
